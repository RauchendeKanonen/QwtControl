Drawing the response of a Control-System


Consider the following transfer-function of the plant:





1. Open QwtControl and insert the expression with add->expression.
In the Exprssion-Dialog write down the expression.


Place a name for the expression before the expression itself:

Name:K*w0^2/(s^2+2*zeta*w0*s+w0^2)


All C math lib calls, such as sqrt(x), pow(x,y), fabs(x) and so on can be
used. Potences can be written x^y. The imaginary unit has to be expressed
as upper-case I  !!!  Be sure to apply also a input to the transfer-function
such as the step response (1/s).

If the picture below the Line-Edit shows a valid expression, click OK.


2. Add the Variables by using add->Variable.
In the Variable-Dialog type its name and its Value. Then click OK.


Be sure to add K, w0, zeta with its initial values.


3. Do a right-mouse-click onto the expression. In the drop-down-menu
select "draw response". In the following Dialog, select the range and
resolution an s as the depending variable. Then click OK.
A new window opens with the response of the function.


4. Do a right-mouse-click onto the variable zeta. In the drop-down-menu
select "add Slider". A Dialog for the range of the slider opens. For zeta

set the range to 0-1. Then click OK.

A new window with the slider opens.


5. Move the slider by clicking with the mouse and dragging it. Observe
the changing response.


Same as above applies to Closed-Loop transfer-functions.

Drawing the Root-Locus of a Control-System



Consider the following transfer-function of the plant:








1. Solve the characteristic equation for the complex variable s:


s=-w0*sqrt(zeta^2-1)-w0*zeta
s=w0*sqrt(zeta^2-1)-w0*zeta


Note: This expressions are quite simple. The solutions of a third order
characteristic equation can be much bigger.
Usually the output of maxima (cas) can be pasted into the Line-Edit of
the Expression-Dialog.


2. Insert the expression with add->expression.
In the Exprssion-Dialog write down the expression.


Place a name for the expression before the expression itself:

root1:-w0*sqrt(zeta^2-1)-w0*zeta


For the second root:

root2:w0*sqrt(zeta^2-1)-w0*zeta


All C math lib calls, such as sqrt(x), pow(x,y), fabs(x) and so on can be
used. Potences can be written x^y. The imaginary unit has to be expressed
as upper-case I  !!! 

If the picture below the Line-Edit shows a valid expression, click OK.


3. Add the Variables by using add->Variable.
In the Variable-Dialog type its name and its Value. Then click OK.


Be sure to add w0, zeta with its initial values. Kp must not be added,
since it will be defined as the independent Variable of the locus's.


4. Do a right-mouse-click onto the expressions. In the drop-down-menu
select "draw root locus". In the following Dialog, select the range and
resolution and Kp as the depending variable. Then click OK.
The new curves are drawn in complex plane. You can see
the root-locus of the plant, depending on Kp. A red filled circle
shows the root-location of the transfer-function, A X-cross shows
the pole-location at the beginning of the interval of Kp. An
a cross shows the position of the pole at actual Kp.


5. In the independent Variable-Slider-Dialog move the slider for Kp
and observe the actual pole location change.


Same as above applies to Closed-Loop transfer-functions.


Look this video for drawing a step response.
